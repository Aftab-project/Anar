## 🔄 Future Updates

When adding new content to your website:

### Do's ✅
- Test on mobile devices after making changes
- Use the same responsive patterns already in place
- Keep minimum 44px tap targets for buttons
- Ensure text is at least 14px on mobile

### Don'ts ❌
- Don't add fixed widths without testing on mobile
- Don't make tap targets smaller than 44px
- Don't assume desktop styles work on mobile
- Don't remove viewport meta tag from HTML

## 🆘 Need Help?

### If Something Looks Wrong:
1. **Clear your browser cache** (Ctrl+F5)
2. **Test on actual device** (not just browser DevTools)
3. **Check different browsers** (Safari, Chrome)
4. **Verify viewport tag** is in HTML head section
5. **Review documentation** in this folder

### Common Solutions:
- **Menu not working?** Check JavaScript console for errors
- **Layout broken?** Clear cache and hard refresh
- **Text too small?** Check if viewport meta tag exists
- **Images not loading?** Verify image file paths
